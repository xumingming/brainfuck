# brainfuck

Clojure version of [Brainfuck](http://en.wikipedia.org/wiki/Brainfuck)

## Usage

```bash
$ git clone https://github.com/xumingming/brainfuck.git
$ cd brainfuck
$ echo "++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>." > helloworld.fb
$ lein run helloworld.fb
Hello World!
```
